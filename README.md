# The Smart Ring Lab (Hugo)

This is a minimal Hugo site built for Cloudflare Pages.

## Quick start (local)
1. Install Hugo (extended recommended).
2. Run:
   - `hugo server -D`
3. Open: http://localhost:1313

## Deploy on Cloudflare Pages
- Framework preset: **Hugo**
- Build command: `hugo --minify`
- Build output directory: `public`

Recommended environment variable:
- `HUGO_VERSION` (pick the same version you use locally)

## Content rules (so it doesn't look like AI)
- Start with a direct answer in the first ~100 words.
- Write for search intent: comparison / buying guide / troubleshooting.
- Use short sentences. Avoid filler.
- Add FAQ (8–15 questions) to capture long-tail queries.
- Always link back to hubs:
  - /best-smart-rings/
  - /ringconn/
  - /ringconn-vs-oura/
